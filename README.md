# auto-maroc# موقع سيارات مستعملة في المغرب

موقع لبيع وشراء السيارات المستعملة في مختلف مدن المغرب.

## المميزات
- تصفح السيارات حسب المدينة
- فلترة حسب الماركة والسعر
- إضافة إعلانات جديدة
- تواصل مباشر بين البائع والمشتري

## التقنيات المستخدمة
- HTML5, CSS3, JavaScript
- GitHub Pages للاستضافة

## كيفية المساهمة
1. Fork المشروع
2. إنشاء فرع جديد
3. إضافة التعديلات
4. عمل Pull Request
